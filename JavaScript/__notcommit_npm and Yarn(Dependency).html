# npm and Yarn(Dependency)

# What is npm(Yarn)?

  • 일단 자바스크립트가 무엇인지 부터 알아야 겠다. 자바스크립트는 태생이 웹페이지를 돕기 위해 만들어진 것으로써, 공식적으로 모듈 사용(Python: import, C: #include "foo.h")이 지원이 되지 않는다고 한다. 모듈화를 위해 자바스크립트에 CommonJS, AMD라는 모듈 정의 규칙이 생겨났다고 한다.
  • 이러한 모듈을 쓰기 좋게 정리해놓은 것이 패키지다. 그 패키지의 설치, 버전 관리를 도와주는 툴이 바로 npm(또는 Yarn)이다.

# npm의 사용법

  • package-name 패키지 설치

npm install package-name

  • 특정한 프로젝트가 아닌 모든 프로젝트에 패키지를 적용하고 싶은 경우

npm install -g package-name

package.json 파일의 "dependencies"를 보자. 이 프로젝트가 의존하고 있는 패키지의 이름과 버전들이 나와있다. 이제 이 파일만 있으면 집 앞의 아무곳이나 가서

npm install 를 입력하면, 내가 필요로하는 프로젝트의 패키지를 아주 빠르게 설치 할 수 있다.

패키지의 원하는 버전 설치(package-name의 1.1.0버전, Semantic Versioning을 지원한다고 한다.)

npm install package-name1.1.0
